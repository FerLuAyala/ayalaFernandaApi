# 🌌 API REST - Usuarios | Signos | Horóscopos

Esta API permite gestionar **usuarios, signos zodiacales y horóscopos**, con operaciones CRUD (crear, leer, actualizar, eliminar).

---

## 📌 Tecnologías utilizadas
- **Node.js**
- **Express**
- **MongoDB**
- **Bootstrap 5**
- **JavaScript**
