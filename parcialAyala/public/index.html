Rutas de usuario

    GET
  • /api/users Ver lista de usuarios
    • POST
  • /api/users Guardar un usuario nuevo 
                   
    Ejemplo
              
        {
            "name": "Marcelo Suarez",
            "fechaNac": "09-05-1960",
            "email": "marcelo@example.com",
            "password": "1236",
        }
    • PUT
  • /api/users/:id Modificar usuario 
                    
    Ejemplo
                                     
        {
            "name": "Nombre-apellido",
            "fechaNac": "xx-xx-xxxx",
            "email": "mail@example.com",
            "password": "xxxxx",
        }
    • DELETE
  • /api/users/:id Selecciona el id que quieras eliminar
