API Documentación

¿Por qué utilizar esta Api?

Esta API de signos del zodíaco te permite consultar información de cada signo, sus características, fechas y predicciones diarias de forma sencilla.

los Link de GET estan cargados, salvo el de signos por fecha

Rutas de Usuarios

    GET
  • /api/users
    Ver lista de usuarios
    /api/users/:id
    Ver usuarios id
    • POST
  • /api/users
    Guardar un usuario nuevo 
                   
    Ejemplo
              
        {
            "name": "luis Suarez",
            "fechaNac": "09-05-1960",
            "email": "luis@mail.com",
            "password": "12w6"
        }
    • PUT
  • /api/users/:id
    Modificar usuario 
                    
    Ejemplo
                                     
        {
            "name": "Jorge Suarez",
            "fechaNac": "03-08-1960",
            "email": "jorge@mail.com",
            "password": "45681",
        }
    • DELETE
  • /api/users/:id
    Selecciona el id que quieras eliminar

Rutas de Signos

    GET
  • /api/signos
    Ver lista de signos
    /api/signos/buscar/:nombre
    Ver signos por nombre
    /api/signos/:id
    Ver signos por ID
    /api/signos/signo/
    Ver signos por Fecha 
    Ejemplo
 {
    "dia": 09,
    "mes": 01
 }
    • POST
  • /api/signos
    Guardar un signo nuevo 
                   
    Ejemplo
         
    {     
      "name": "Alguno",
      "elemento": "Tierra",
      "fechaInicio": "2025-04-12",
      "fechaFinal": "2025-05-11",
      "icono": "💟",
      "caracteristicas": "Paciente - leal",
      "descripcion": "Breve texto"
    }
    • PUT
  • /api/signos/:signo
    Modificar Signo 
                    
    Ejemplo: puede modificar uno o varios campos
                                     
{
 "elemento": "Fuego",
 "caracteristicas": "Bueno, Leal", 
  "fechaFinal": "2025-05-16"
}
    • DELETE
  • /api/signos/:id
    Selecciona el id que quieras eliminar

Rutas de Horóscopos

    GET
  • /api/horoscopos
    Ver lista de horóscopos
    /api/horoscopos/:id

    /api/horoscopos/buscar/:name
    Ver horóscopos por nombre del signo
    • POST
  • /api/horoscopos

    Ejemplo

 "name": "Piscis",

}

  • /api/horoscopos/:id

{
 "name": "Piscis",
 "prediccion": "Hola me actualice",
}
    • DELETE
  • /api/horoscopos/:id
    Selecciona el id que quieras eliminar
